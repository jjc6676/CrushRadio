Crush Radio

Open-source radio for songs algorithms would bury. Artists upload originals. Everyone hears the same broadcast. If listeners tap CRUSHED IT, the track survives. If not, it disappears.

The Code // Built in the open

Crush Radio is built in public on GitHub. An open-source community radio station. Every change is a pull request. Every decision is in an issue or a commit message. No closed doors.
