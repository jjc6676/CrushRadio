# Active Context: CrushRadio

## Project Summary
CrushRadio is a modular web application for streaming online radio and YouTube audio, designed for extensibility and maintainability. The project is structured with clear separation between UI, backend endpoints, and helpers.

## Current State
- Core directories (`components`, `endpoints`, `helpers`, `pages`) are present.
- Documentation initialized in `docs/`.
- No authentication or user management yet.

## Next Steps
- Implement core radio and YouTube audio features.
- Add robust error handling in endpoints and UI.
- Plan for user features (playlists, favorites, authentication).

## Key Considerations
- Maintain modularity for future integrations.
- Ensure user-friendly error messages.
- Plan for scalability and performance.
